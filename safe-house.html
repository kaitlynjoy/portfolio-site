Safe House Subleasing

A promotional site for a sublease-finder service.

Visit Site

A Solution to a Problem

I created the concept of Safe House, a sublease-finder after finding out how difficult it is to find short-term housing. My challenge was to position Safe House as a safe, simple service, and provide information and demos to draw viewers to sign up.

Project Features

  • Responsive, custom HTML5 and CSS3 site
  • Original JavaScript/jQuery image gallery
  • Subtle animations
  • Video & maps integration
  • Clean, minimal design

Challenges

Responsive Search Results Layout

Each search result contains quite a bit of information. This information needed to be easily viewed on all device sizes. With the help of media queries and phone, tablet, and desktop views of the page, results can be viewed simply and easily.

Custom Image Gallery

I wanted to create a simple, lightweight image gallery that had a minimalistic layout, allowing the user to switch between images by choosing a thumbnail.
