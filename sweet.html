Sweet Fruit Water

A short, informational promo for a fun new product.

Storytelling Through Interactivity

Sweet Fruit Water is a (fictional) product, about to be released to the market. This horizontal-scrolling website engages the user and promotes the product.

Project Features

  • Horizontal Scrolling
  • Full-Screen Layout
  • Logo Design & Branding
  • Interactivity with Javascript & jQuery
  • Unobtrusive Image Fader

Challenges

Interactive Functionality

I was inspired to create a site with horizontal scrolling when I came across Viget’s Careers mini-site. I customized a version of Alvaro Trigo’s fullPage.js

Background Layering

Using CSS3’s multiple backgrounds, I created a three dimensional look, layering the subtle texture against a version of the logo, with the content on top.

Dynamically Switching Flavors

Using custom JavaScript and jQuery, I created a content-switcher to navigate between information about each flavor. See it in action here.
